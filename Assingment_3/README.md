# BLN_Assignments
Introduction to Block Chain and Cryptocurrency Course Assignments
___

### Installation
- Copy the code from `.sol` file.
- Open [Remix IDE](http://remix.ethereum.org/)
- Paste the code and deploy the smart contract.
