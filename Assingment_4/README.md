# BLN_Assignments
Introduction to Block Chain and Cryptocurrency Course Assignments
___

### Installation
- Run the `server` as `python server.py`.
- Run the `client` as `python client.py <server IP address>`.
- Note: `P3` and `P4` are incomplete.
